# ✦ Sparkles Travel Group — Website & Community

A modern, sleek, and futuristic travel agency website for **Sparkles Travel Group**.

## 🌐 Live Site

Coming soon at [sparklestravelgroup.com](https://sparklestravelgroup.com)

## 🚀 Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — No framework dependencies, lightweight & fast
- **Font Awesome** — Icon library
- **Google Fonts** — Inter + Playfair Display

## ✨ Features

- **Hero Section** — Animated particles, gradient backgrounds, stats counter
- **Destinations** — Grid gallery with hover effects
- **Services** — 6 service cards with glow animations
- **Booking Form** — Full trip request form with validation
- **Appointment Calendar** — Placeholder for Calendly/Google Calendar integration
- **Community Hub** — Forums, stories, deals, events + email signup
- **About Section** — Company story, values, partnership badges
- **Testimonials** — Client reviews carousel
- **Contact** — Email, scheduling, social media links
- **Responsive** — Fully mobile-optimized (mobile menu, stacked layouts)
- **Preloader** — Branded loading animation
- **Scroll Animations** — Reveal-on-scroll for all sections
